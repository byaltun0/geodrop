## GitHub Pages kurulumu

1. GitHub'da yeni repo oluştur (ör. `geodrop-website`)
2. Bu klasörün **tüm içeriğini** repo köküne yükle
3. Settings → Pages → Source: **Deploy from branch**
4. Branch: `main` / Folder: **/ (root)**
5. Birkaç dakika sonra site yayında:
   - `https://KULLANICI.github.io/geodrop-website/tr/`
   - veya özel domain: `geodrop.app`

## Play Console URL'leri
- Gizlilik: `https://geodrop.app/tr/privacy.html` (veya GitHub Pages URL)
- Koşullar: `.../tr/terms.html`
- Hakkında: `.../tr/about.html`

Admin panelde Yasal → Görünüm: **LİNK** seç → URL'leri gir.
